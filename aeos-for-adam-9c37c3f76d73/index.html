02 / 07 The shape of it One paragraph version

Claude Code is the harness. Postgres + pgvector is the brain. Skills route intent. Agents do the work. MCP servers are the hands.

03 / 07 The brain Where everything lives

One table. Typed nodes. Typed edges. Every entity is searchable.

A person, a task, a bill, an email, a research claim, a decision — all the same primitive: a row in brain_node with a node_type column and a JSONB metadata blob. Relationships are brain_edge rows with a typed edge_type. Every node is embedded with OpenAI text-embedding-3, so semantic search crosses CRM, finance, research, and todos in one query.

04 / 07 Skills + agents How routing works

Skills trigger semantically. Agents do the actual work.

I don't memorize commands. I ask in English. The right skill activates, loads its reference library into context, and either answers directly or spawns specialist agents. Nine skills cover everything; thirty-four agents handle specific operations (read-only, write, sync, audit).

05 / 07 Architecture Data flow

Three loops: ingest, query, act.

Ingest

Capture

Email sync (Gmail + Outlook), Plaid txn sync, scanned mail via vision, URL ingestion, conversation learnings, Jira webhooks. Each source becomes typed brain nodes with provenance.

→
Store

Embed + link

pgvector embeddings on every node. Edges proposed automatically (depends_on, relates_to, contradicts, supports, belongs_to, follow_up). Confidential flag for sensitive content.

→
Act

Specialist agents

Skill routes intent → agent picks tools → confirms before consequences (sending email, creating tickets, modifying shared state) → writes audit row.

M

MCP servers

Five custom servers expose brain, audit, core (tenant), email-sync, and research-sources to Claude. JWT-gated, service-role-backed.

F

Edge Functions

Deno-runtime workers on Supabase: signal classifier, Plaid sync, MS Graph proxy with device-code auth + token refresh, document vault.

S

Security model

Anon key has zero read access. All Edge Functions use service_role. Tokens in Vault. Confidential nodes flagged. All repos private.
